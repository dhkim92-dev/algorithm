# 백준 35425 Split the GSHS 5

## 문제 접근

이 문제는 정해진 C 의 용량 내에 몇명을 태워 이동시키는 문제로 
파티셔닝 문제에 해당함.  

거기에 최소 시간 이라는 힌트가 있듯 최적화 문제에 해당하며  
N은 300,000이라는 큰 수로 O(N^2) 의 시간복잡도만 되어도 시간초과  

시간 순 정렬이 보장되어 있어 이분 탐색에 의한 최적화를 생각할 수도 있지만 
구간을 구하는 것이 아니라서 사용 불가능.  

태우는 순서를 바꿀순 없기 때문에 이는 DP에 의한 최적화로 문제를 풀어야한다고 추정해야한다.
